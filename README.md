# 🍅 Tomato Disease Detection using Deep Learning

## 📌 Project Overview
This project is a deep learning-based image classification system designed to detect tomato plant diseases from real-field images. The model is trained using a Convolutional Neural Network (CNN) and helps identify different tomato leaf diseases automatically.

---

## 🚀 Features
- Tomato leaf disease classification
- Real-field image dataset support
- Deep learning-based CNN model
- Dataset preprocessing and augmentation
- Model training and evaluation
- Google Colab compatible

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- OpenCV
- Scikit-learn
